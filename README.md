A simple and responsive web application that calculates your exact age in years, months, and days based on your date of birth.

Click here to view the App -> https://priyankanagalkar.github.io/Age-Calculator-JS/

- Calculate exact age in years, months, and days
- Clean and responsive design
- Input validation for invalid dates
- Works on all devices (mobile, tablet, desktop)

Built With
- HTML5
- CSS3
- JavaScript (Vanilla)
